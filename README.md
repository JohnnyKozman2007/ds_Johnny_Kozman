
# Data Science Internship Assignment - Johnny Kozman

## Project Overview
Analysis of the relationship between trader behavior and market sentiment using:
- Bitcoin Fear & Greed Index data
- Historical trader data from Hyperliquid

## Files Structure
- `notebook_1.ipynb` - Complete analysis with EDA, visualizations, and Random Forest model
- `csv_files/` - Directory for data files
- `outputs/` - Contains all generated charts and visualizations
- `ds_report.pdf` - Summary report (to be downloaded separately)

## Key Findings
- Identified correlation between market sentiment and trading profitability
- Built predictive model using Random Forest classification
- Provided actionable trading strategy recommendations

## Setup
All analysis was conducted in Google Colab with Python 3.7+
